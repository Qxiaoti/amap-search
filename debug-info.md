# 调试指南

## 如何查看控制台错误

1. 打开浏览器（Chrome/Edge）
2. 按 F12 打开开发者工具
3. 切换到 "Console" 标签
4. 查看红色错误信息

## 常见错误及解决方案

### 1. "Cannot read property of undefined"
- 原因：数据还未加载
- 解决：已添加空值检查

### 2. "Failed to fetch"
- 原因：网络问题或API Key错误
- 解决：检查网络连接和API Key

### 3. "AbortError"
- 原因：请求被取消（正常）
- 解决：无需处理

### 4. "请求超时"
- 原因：网络慢或API响应慢
- 解决：重试或检查网络

## 性能监控

控制台会显示：
- "开始搜索..." - 搜索开始
- "API响应时间: XXXms" - API响应速度
- "总耗时: XXXms" - 完整搜索时间

## 如果还有问题

请复制控制台的完整错误信息，包括：
- 错误类型
- 错误消息
- 堆栈跟踪（Stack trace）
