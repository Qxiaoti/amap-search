## 📝 测试步骤

### 基本流程测试

1. 打开应用 → 进入设置 → 输入API Key
2. 进入搜索页 → 输入"广告" → 选择城市 → 搜索
3. 查看结果 → 复制电话 → 添加收藏
4. 导出Excel → 检查文件
5. 查看收藏 → 导出收藏
6. 查看历史 → 重新搜索
7. 查看分析 → 检查统计

### 异常流程测试

1. 不输入API Key → 应显示错误
2. 不输入关键词 → 应显示错误
3. 搜索不存在的内容 → 应显示"未找到"
4. 网络断开 → 应显示"请求失败"
5. 快速点击搜索 → 应取消旧请求
