# 渐变动态按钮更新说明

## 更新时间
2026年2月13日

## 新增功能

### 1. 全局渐变按钮样式
在 `src/style.css` 中添加了全局渐变按钮类 `.gradient-btn`

**特效包括：**
- 🌈 **渐变背景**：粉色系渐变（#ff6482 → #ff8fab → #ffa8c5）
- ✨ **动态流动**：背景渐变持续流动动画（3秒循环）
- 💫 **光泽效果**：鼠标悬停时有光泽扫过效果
- 🎯 **脉冲动画**：悬停时按钮轻微放大缩小
- 🔆 **阴影增强**：悬停时阴影更明显
- 👆 **点击反馈**：点击时按钮缩小效果

### 2. 应用渐变效果的按钮

#### Search.vue（搜索页面）
- ✅ 主搜索按钮
- ✅ 确定位置按钮
- ✅ 城市选择确认按钮

#### Home.vue（首页）
- ✅ 开始搜索按钮

#### Favorites.vue（收藏页面）
- ✅ 去搜索按钮

## 技术实现

### CSS动画
```css
@keyframes gradient-shift {
  0% { background-position: 0% 50%; }
  50% { background-position: 100% 50%; }
  100% { background-position: 0% 50%; }
}

@keyframes button-pulse {
  0%, 100% { transform: scale(1); }
  50% { transform: scale(1.02); }
}
```

### 渐变配置
- **背景渐变**：135度线性渐变
- **背景尺寸**：300% × 300%（用于动画流动）
- **动画时长**：3秒（正常）/ 1.5秒（悬停加速）
- **光泽效果**：白色半透明层从左到右扫过

### 禁用状态处理
- 禁用时自动移除渐变和动画
- 显示灰色背景（#d2d2d7）
- 鼠标样式变为不可点击

## 使用方法

### 给按钮添加渐变效果
只需在按钮上添加 `gradient-btn` 类：

```vue
<button class="my-btn gradient-btn">按钮文字</button>
```

### 禁用状态
```vue
<button class="my-btn gradient-btn" :disabled="loading">
  {{ loading ? '加载中...' : '确定' }}
</button>
```

## 视觉效果

### 正常状态
- 粉色渐变背景持续流动
- 柔和的阴影

### 悬停状态
- 渐变流动速度加快（1.5秒）
- 按钮轻微脉冲（放大1.02倍）
- 光泽从左到右扫过
- 阴影增强并上移2px
- 阴影颜色：rgba(255, 100, 130, 0.4)

### 点击状态
- 按钮缩小到0.98倍
- 回到原位（无上移）

### 禁用状态
- 灰色背景
- 无动画
- 无阴影
- 鼠标显示禁止图标

## 性能优化

1. **GPU加速**：使用 transform 而非 position 进行动画
2. **will-change**：可选添加以提示浏览器优化
3. **动画暂停**：禁用时停止所有动画
4. **合理的动画时长**：避免过快或过慢

## 兼容性

- ✅ Chrome/Edge（完美支持）
- ✅ Firefox（完美支持）
- ✅ Safari（完美支持）

## 注意事项

1. 不要在危险操作按钮上使用渐变效果（如删除、清空）
2. 禁用状态会自动移除渐变
3. 小按钮可以调整padding以适配渐变效果
4. 确保按钮文字颜色为白色以保证可读性
