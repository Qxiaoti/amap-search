# 城市选择器渐变效果更新

## 更新时间
2026年2月13日

## 更新内容

### 1. 热门城市按钮
**位置**: 城市选择弹窗 → 热门城市标签

**效果**:
- 默认：浅灰色背景 (#f5f5f7)
- 悬停：粉色渐变背景 + 白色文字 + 上浮2px + 阴影
- 渐变：135度线性渐变 (#ff6482 → #ff8fab)

### 2. 城市标签（热门/按省份）
**位置**: 城市选择弹窗顶部标签

**效果**:
- 默认：透明背景 + 灰色文字
- 悬停：浅灰色背景
- 激活：粉色渐变背景 + 白色文字 + 流动动画
- 渐变动画：3秒循环流动

### 3. 省份/城市/区县列表项
**位置**: 三级联动选择列表

**效果**:
- 默认：透明背景
- 悬停：淡粉色渐变背景 + 右移4px + 轻微阴影
- 激活（已选中）：粉色渐变背景 + 白色文字 + 流动动画
- 悬停渐变：135度线性渐变 (#ffe8ed → #fff0f3)
- 激活渐变：135度线性渐变 (#ff6482 → #ff8fab)

### 4. 搜索结果项（城市搜索）
**位置**: 城市选择弹窗 → 搜索框下方的结果列表

**效果**:
- 默认：浅灰色背景
- 悬停：粉色渐变背景 + 白色文字 + 右移4px + 阴影
- 渐变：135度线性渐变 (#ff6482 → #ff8fab)

### 5. 加载更多按钮
**位置**: 搜索结果底部

**效果**:
- 应用全局渐变按钮样式
- 流动渐变动画
- 悬停时脉冲效果
- 光泽扫过效果

## 视觉层次

### 一级交互（主要操作）
- 确认按钮
- 搜索按钮
- 加载更多按钮
- **效果**: 完整渐变动画 + 脉冲 + 光泽

### 二级交互（选择操作）
- 热门城市按钮
- 搜索结果项
- **效果**: 悬停渐变 + 位移 + 阴影

### 三级交互（导航操作）
- 标签切换
- 省份/城市/区县选择
- **效果**: 激活时渐变动画，悬停时淡色背景

## 颜色方案

### 主渐变色
```css
linear-gradient(135deg, #ff6482, #ff8fab)
```

### 淡渐变色（悬停）
```css
linear-gradient(135deg, #ffe8ed, #fff0f3)
```

### 阴影颜色
```css
rgba(255, 100, 130, 0.3)  /* 标准阴影 */
rgba(255, 100, 130, 0.15) /* 轻微阴影 */
```

## 动画时长

- 标准过渡：0.3s
- 渐变流动：3s（循环）
- 悬停加速：1.5s（仅主按钮）

## 交互反馈

### 位移反馈
- 上浮：主按钮悬停 (-2px)
- 右移：列表项悬停 (+4px)

### 缩放反馈
- 脉冲：主按钮悬停 (1.0 ↔ 1.02)
- 点击：主按钮点击 (0.98)

### 阴影反馈
- 轻微：列表项悬停
- 标准：按钮悬停
- 增强：主按钮悬停

## 一致性设计

所有城市选择相关的交互元素现在都遵循统一的视觉语言：

1. **粉色系渐变**：品牌色一致性
2. **流畅动画**：0.3s标准过渡
3. **层次分明**：不同级别不同效果强度
4. **反馈明确**：位移、阴影、颜色变化

## 用户体验提升

1. **视觉吸引力**：渐变效果更现代、更有活力
2. **交互反馈**：每个操作都有明确的视觉反馈
3. **品牌一致性**：所有按钮风格统一
4. **层次清晰**：主次操作通过效果强度区分
5. **流畅体验**：动画时长适中，不会感觉卡顿

## 性能考虑

- 使用 CSS transform 而非 position 进行动画
- 渐变动画仅在激活状态下运行
- 悬停效果使用简单的渐变，不消耗过多资源
- 所有动画都有合理的时长，避免过度渲染
